Weatherify

Front End

Weatherify

Weather Application

Tech Stack: React, Moment & Moment-Timezone, react-bootstrap, Sass

Created as an ongoing and iterative project, the Weatherify App allows users to check the current daily and weekly forecast of almost every city around the world. The OpenWeather API is being used to display data such as temperature, wind speed and direction, and humidity; while all icons are provided by the Weather Icons library.

  • Used Axios to fetch daily and weekly weather data from the api and combined in it into one call using the .all() method.
  • Using a timezone switch statement, I created a method that combines Moment and Moment-Timezone to dynamically change the background, application title, and icons based on the UTC Time, timezone, and weather condition code of the city as reported by the API.
  • Styled with Sass, the application uses bootstrap and react-bootstrap. Furthermore, all gradient backgrounds have been made using Adobe Illustrator
  • As the development of the application progress', I hope to improve on existing features, on the code itself by making it DRY, and to add more advanced features and styling. Please see the projects github repo below for more information.

CyberPunk Mud

Full Stack

CyberPunk Mud

Multi-User Dungeon Game

Tech Stack: React, Redux, React-Vis, Sass, Python, Django, PostgreSQL

I and four other students developed a simple interactive, multi-user game during the Computer Science Build Week at Lambda. Consisting of 101 rooms, users can explore this retro cyberpunk-inspired game and see other users in real-time.

  • With only one day to learn Django, we deployed a RESTful API backend to Heroku and used PostgreSQL for data persistence.
  • An algorithm was implemented on the backend to generate a spiral maze with which users can interact with and explore.
  • Styled with Sass, the React/Redux frontend uses React-vis to display the maze in a spiral node like fashion.
  • I took on the role as the UX/UI designer and created/implemented the retro-cyberpunk aesthetic of the game using Figma to create a moodboard and wireframes.

Sleep Tracker Application

Tech Stack: React, React-vis, React-big-calendar, Formik, Styled Components, fontawesome

The Sleepmood application was built by a team of one UX designer,three UI, three frontend, and one backend developer(s). The React app enables users to analyze and repair their sleeping habits based purely on their moods during the day, before they go to sleep, and after.
