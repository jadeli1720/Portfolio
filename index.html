CyberPunk Mud

Full Stack

CyberPunk Mud

Multi-User Dungeon Game

Tech Stack: React, Redux, React-Vis, Sass, Python, Django, PostgreSQL

I and four other students developed a simple interactive, multi-user game during the Computer Science Build Week at Lambda. Consisting of 101 rooms, users can explore this retro cyberpunk-inspired game and see other users in real-time.

  • With only one day to learn Django, we deployed a RESTful API backend to Heroku and used PostgreSQL for data persistence.
  • An algorithm was implemented on the backend to generate a spiral maze with which users can interact with and explore.
  • Styled with Sass, the React/Redux frontend uses React-vis to display the maze in a spiral node like fashion.
  • I took on the role as the UX/UI designer and created/implemented the retro-cyberpunk aesthetic of the game using Figma to create a moodboard and wireframes.

Riders For Life

Full Stack

Riders For Life

Admin Dashboard & Medical Emergency SMS System

Tech Stack: React, Redux, FrontlineSMS, Node, Express, GeoLib, PostgreSQL

The 'Riders for Life' project was a team collaboration consisting of 1 UX Designer and 6 other Web Developers. As part of Lambda Labs, the goal of the project was to create a two-part application for the non-profit Safe Mothers, Safe Babies in Uganda.

  • Working with the non-profitsâ€™ stake-holder for 8 weeks, we increased the productivity, reliability, and usability of both parts of the application; enhancing the overall quality of the users experience.
  • My primary focus, was building the database for our node backend, and integrating Frontline SMS. This text-messaging cloud service provided a reliable SMS emergency ride request system for pregnant mothers and Boda drivers in Uganda.
  • We employed the use of the GeoLib library to help the application locate the nearest driver to a pregnant mother requesting a ride, and then dispatched them to said mother for emergency services.
  • A React/Redux administrative dashboard was developed to allow administrative users to interact with mother/driver data collected on the ground in Uganda.

African Marketplace

Admin Dashboard & Medical Emergency SMS System

Tech Stack: Node, Express, Knex, Helmet, Supertest, SQLite3, PostgreSQL

A week-long team collaboration, the African Marketplace App allows African merchants to upload products for sale and view the prices offered by other merchants.

  • As the backend developer, I created the Heroku deployed RESTful API that is used by the React frontend application. Documentation can be found here and was created using Postman.
  • Using Node and Express, endpoints were created for a collection of products and relevant pricing data, allows CRUD operations to be performed on said data, and allows for users to register and log in using protected routers and hashed password encryption.
  • During the development phase of the project, the database was created using Knex and SQlite3 but uses PostgreSQL in the production server.

sleepmood

Sleep Tracker Application

Tech Stack: React, React-vis, React-big-calendar, Formik, Styled Components, Fontawesome

The Sleepmood application was built by a team of one UX designer,three UI, three frontend, and one backend developer(s). The React app enables users to analyze and repair their sleeping habits based purely on their moods during the day, before they go to sleep, and after.

  • As one of the front-end developers, I helped create forms using a combination of the Formik library and fontawesome emojis'; allowing users to easily and quickly record their sleep mood data.
  • On the users dashboard, a weekly overview of a thier sleep mood data was provided and implemented with the React-vis library; a react data-visualization library developed by Uber.
  • To fit the application's overall aesthetic theme created by our UX designer, React-big-calendar was customized to allow users to view and edit their sleep mood data from a monthly calendar overview.
